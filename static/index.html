Bộ Công Cụ SentinX

Ứng dụng quét lỗ hổng web tự động cho pentester.

Giải Thích Ma Trận Lỗ Hổng

SentinX thu thập mục tiêu web, ánh xạ các tuyến request và thực hiện kiểm tra lỗ hổng trên các lớp bảo mật sau:

Kiểm tra Header Bảo Mật

Kiểm tra việc triển khai Content Security Policy (CSP), HSTS, X-Frame-Options, X-Content-Type-Options và Referrer-Policy để giảm rủi ro framing, MIME-sniffing và rò rỉ dữ liệu.

Kiểm tra SSL/TLS

Kiểm tra tính hợp lệ của chứng chỉ, hạn dùng, thông tin CA và cấu hình mã hoá.

Quét cổng (Nmap)

Xác định cổng đang lắng nghe và dò thông tin dịch vụ để phát hiện cấu hình sai hoặc điểm yếu.

Enumerate Thư Mục

Liệt kê thư mục và tệp có thể truy cập bằng danh sách từ khóa để tìm cổng ẩn, bản sao lưu cấu hình hoặc file log.

Phát hiện SQL Injection

Phân tích endpoint GET/POST, thử các payload trên tham số để tìm điểm có thể bị tiêm SQL.

Cross-Site Scripting (XSS)

Chèn các vector HTML/JS tiêu chuẩn vào input để kiểm tra phản chiếu trong phản hồi và đánh giá nguy cơ thực thi.

Environment Information

Backend Framework: FastAPI v0.139+
Server Engine: Uvicorn on Python 3.13
Web Client: Vanilla ECMAScript & CSS Grid
Port Scan Driver: Python-nmap binding wrapper
SQL Injection Agent: Dynamic injection modules
