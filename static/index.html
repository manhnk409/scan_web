SentinX Scan Suite

An automated web vulnerability scanning application designed for penetration testers.

Vulnerability Matrix Explanations

SentinX crawls web targets, maps request routes, and performs structured vulnerability audits on the following security layers:

Security Headers Audit

Verifies deployment of Content Security Policy (CSP), Strict-Transport-Security (HSTS), X-Frame-Options, X-Content-Type-Options, and Referrer-Policy to prevent framing, MIME-sniffing, and data leaks.

SSL/TLS Protocol Testing

Examines certificate validity, dates, CA issuer metadata, and tests compliance with standard encryption configurations.

Nmap Port Mapping

Identifies active server ports and queries network services for version signatures to reveal misconfigured or vulnerable endpoints.

Directory Brute Force

Enumerates accessible subdirectories and source files using target lists to locate hidden portals, configuration backups, or logs.

SQL Injection Discovery

Analyzes GET and POST endpoints, testing query parameters with active payloads to map vulnerable database connectors.

Cross-Site Scripting (XSS)

Injects standard HTML/JS vectors into discovered inputs, verifying reflection context in responses to determine execution risks.
